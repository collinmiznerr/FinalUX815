09

What's in a Name?

Built a random name generator using 11ty and Alpine.js. A button click picks a random name from a list of ten and displays it using Alpine's reactive data binding to update the DOM without writing any manual JavaScript event listeners. A CSS framework was used to style the interface.

Live Preview

Open Lab

Skills Used

Alpine.js 11ty JavaScript External JS libraries

Key Takeaways

Alpine.js handles reactivity directly in HTML attributes.

Keeping logic like the name list and random number function in a separate JS file keeps the HTML clean and easier to maintain.

Process

1

Added Alpine.js to the <head> of the main layout template using a CDN <script defer> tag

2

Created a separate JS file with a list of 10 names and a function to pick one at random using Math.random()

3

Built the HTML interface an <h1> to Alpine's reactive data and a button wired to the picker function x-on:click.

4

Tested the button interaction and verified the name updates correctly on each click.

Challenges

The hardest part was figuring out how Alpine.js accesses functions and data defined outside of the HTML file. I had to make the name list and random picker function available on the window object so Alpine's x-data could reference them.
