07

Explorer's Blog

Built a multi-page static blog using 11ty, written from the perspective of a lone interstellar explorer. Blog posts were written in Markdown and generated using AI, with a shared layout template powering the header and footer across all four pages: homepage, two blog posts, and about page.

Live Preview

Open Lab

Skills Used

11ty Liquid templates Layouts Markdown Static site generation

Key Takeaways

11ty layouts let you define the HTML shell once and reuse it across every page.

Markdown makes writing content fast and clean. 11ty makes it into valid HTML at build time without extra effort.

Shortcodes are a powerful way to create reusable content blocks that can be dropped into any page with a single line.

Process

1

Set up a new 11ty project and configured the folder structure for layouts, pages, and posts.

2

Built a custom shortcode for a reusable content block

3

Used an AI tool to generate two blog posts written from an interstellar explorer's perspective (cited the model and prompt as an HTML comment)

4

Wrote each blog post as a .md Markdown file.

5

Wrote 10+ CSS rulesets to style the full site.

Challenges

Getting 11ty's layout to work correctly took some trial and error. Understanding how front matter in Markdown files connects to the layout file via the layout key.
