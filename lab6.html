06

The Big Scroll

Built a single-column photo website where each image animates into view as the user scrolls using the ScrollReveal JavaScript library. Each of the five images has its own unique ScrollReveal instance with different animation options.

Live Preview

Open Lab

Skills Used

JavaScript ScrollReveal.js External JS libraries HTML

Key Takeaways

JavaScript load order matters by placing <script> tags at the bottom of <body> ensures the DOM is ready before the script runs.

Third-party libraries like ScrollReveal can add complex interactivity with just a few lines of code.

Process

1

Built the single-column HTML layout with 5 images sized at approximately 500x500px using CSS

2

Added the ScrollReveal library via CDN <script> tag, placed before the closing </body>

3

Read through the ScrollReveal docs to understand options: origin, distance, duration, and delay

4

Tested the scroll behavior in the browser and adjusted timing on each instance.

Challenges

Getting each instance to feel truly distinct took some experimentation with the docs to find parameters beyond the basic defaults.
