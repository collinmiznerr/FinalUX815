Polaroid Cloud

Recreated a multi-section product page for "Polaroid Cloud". A fictitious Polaroid photo storage service, as closely as possible from a provided Figma comp. The focus was on writing reusable HTML and CSS using custom properties to keep styles consistent across repeated components.

Live Preview

Open Lab

Skills Used

HTML CSS Reusable Components CSS Custom Properties

Key Takeaways

CSS custom properties like --color-primary make it easy to update a value once and have it apply everywhere.

Recreating a real design from a comp builds muscle memory for translating visuals into CSS rules.

Process

1

Studied the Polaroid Cloud Figma comp to identify repeating sections and shared styles.

2

Defined CSS custom properties at the :root level for colors, fonts, and sizing.

3

Built the HTML structure section by section, using shared class names for repeated components.

4

Styled each component using the custom properties. Keeping CSS consistent and minimal.

Challenges

The biggest challenge was figuring out how to structure repeated sections (like feature cards) without duplicating CSS. Using a shared class and CSS custom properties helped solve this.
